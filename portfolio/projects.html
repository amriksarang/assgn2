Adobe AEM & Ecommerce Integration Prototype

  • Designed and implemented prototype solution that demonstrated the integration of Adobe AEM's Ecommerce framework with Elasticpath Ecommerce software.

  • The solution used AEM Product Catalog tool to create Product Categories and hierarchy in AEM. The catalog was managed in Elasticpath and product page and media assets were managed in AEM.

  • The AEM ecommerce framework was configured to connect with Elasticpath. Then AEM APIs were used to request data from Elasticpath services.

    • Technologies used:
    Adobe AEM 6.0 CMS, OSGi, Java, JSP, JavaScript, Elastipath ECommerce Software

Feature updates of various websites using AEM CMS

  • Created reusable CMS components and page templates such as headers, footers, content sections, extracting data from data stores and web services, searching content in AEM data sources, process online forms etc.

  • The publishers can add these components to webpages using AEM GUI, author webpage data using Component's GUI, edit webpages directly on the website on dev servers, include DAM assets and publish content to production servers.

Adobe AEM 6.0 CMS, OSGi, Java, JSP, JavaScript

POC demonstrating use of Adode AEM Adaptive form application

  • Created an application that customized AEM Adaptive Forms so that website visitors can submit online requests by filling forms provided on an organization's website.

  • After the online forms were submitted a workflow was initiated in the backend system. The workflow published form data to AEM data stores.

  • The workflow also initiated an approval workflow for the organization's Administrators who could then view the user data and process it further and take decisions as per their organization's procedure.

  • The workflow also intimated the users about the progress of their requests. Adaptive forms facilitate creation of such an application

Technologies used:
Adobe AEM 6.0 CMS, OSGi, Java, JSP, JavaScript
